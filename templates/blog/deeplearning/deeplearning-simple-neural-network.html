Simple Neural Network with TensorFlow 2.8 (Link to GitHub)

The idea

This project was an attempt to construct a simple Neural Network

How it works?

Yo must insert in the field the number of particles that you wanna render. Then, click "enter"

In the dropdown menu, select the interaction and click in simulate. The fastest particle, more red it is and the darkest are slower than the red one.

Skills developed

In order to make the interface of the program, I used Tkinter, and Object Oriented Programming was used. Some basic knoledge of physics were used, like electrostatics and integration. The velocities are inserted into the object function move of Tkinter. This project reinforce my knowledge in basic Python. If you wanna see the source code in GitHub, you can click here
